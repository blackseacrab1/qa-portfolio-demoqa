# Как запустить Postman-коллекцию

Эта коллекция содержит API-тесты для публичного сервиса [JSONPlaceholder](https://jsonplaceholder.typicode.com) (не требует регистрации или API-ключа).

## Требования
- Установленный [Postman](https://www.postman.com/downloads/) (бесплатная версия)

## Шаги

1. Скачайте файл коллекции:  
   [jsonplaceholder-api-tests.postman_collection.json](jsonplaceholder-api-tests.postman_collection.json)

2. Откройте Postman

3. Нажмите Import (в левом верхнем углу)

4. Выберите Upload Files → укажите скачанный .json-файл

5. Коллекция появится в разделе Collections

6. Чтобы запустить все тесты:
   - Нажмите на значок ▶️ Run рядом с коллекцией
   - Нажмите Run

7. Готово! Вы увидите отчёт с пройденными тестами.

> Все запросы используют публичный API и работают без интернет-ограничений.
