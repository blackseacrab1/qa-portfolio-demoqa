# Чек-лист: Text Box, Check Box, Radio Button

## Text Box
- [ ] Поле Full Name принимает буквы (латиница и кириллица)
- [ ] Поле Email принимает любую строку (включая невалидную, например "abc")
- [ ] После нажатия Submit данные отображаются под формой
- [ ] Можно отправить форму с пустыми полями (без ошибок)
- [ ] Поддерживается многострочный ввод в Current Address и Permanent Address

## Check Box
- [ ] Дерево элементов (Home) можно раскрыть/свернуть
- [ ] Можно выбрать один элемент (например, Documents)
- [ ] Можно выбрать несколько элементов (Documents + Downloads)
- [ ] Выбор отображается внизу страницы: "You have selected : ..."
- [ ] Можно снять все флажки — сообщение исчезает

## Radio Button
- [ ] Доступны опции: Yes, Impressive, No (No неактивна)
- [ ] Можно выбрать только одну активную опцию
- [ ] Выбор отображается под кнопками: "You have selected Yes"
- [ ] Невозможно выбрать опцию "No" (disabled)
